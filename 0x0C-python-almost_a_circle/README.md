# 0x0C. Python - Almost a circle

This project is a training to the AirBnB project. The following concepts
where reviewed:

- Import
- Exceptions
- Class
- Private attribute
- Getter/Setter
- Class method
- Static method
- Inheritance
- Unittest
- Read/Write file
